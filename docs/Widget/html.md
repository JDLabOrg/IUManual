## 속성
HTML 위젯은 아래와 같은 속성을 지닙니다.

* 하위요소 : 불가능
* 링크 : 불가능
* 스크롤 애니메이터 : 가능
* 배경이미지 : 가능
* 백엔드 엘립시스 : 불가능
