인풋 텍스트 위젯입니다.<br />
Form 위젯에 추가하여 사용자로부터 텍스트를 입력받을수 있습니다.<br /><br />


## 속성
인풋 텍스트필드 위젯은 아래와 같은 속성을 지닙니다.

* 하위요소 : 불가능
* 링크 : 가능
* 스크롤 애니메이터 : 가능
* 배경이미지 : 가능
* 백엔드 엘립시스 : 불가능
