# 서브밋 버튼 Submit Button ![Submit Button](/../img/widget-submit.png)<br />
![Submit Button Property](/img/property-submit.png)<br /><br />
서브밋 버튼 위젯입니다.<br />
Form에서 입력받은 내용을 전송하는 기능을 담당합니다.
<br /><br />


## 속성
서브밋 버튼 위젯은 아래와 같은 속성을 지닙니다.

* 하위요소 : 불가능
* 링크 : 불가능
* 스크롤 애니메이터 : 가능
* 배경이미지 : 가능
* 백엔드 엘립시스 : 불가능
<br />
